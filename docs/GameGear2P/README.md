# [Game Gear](https://en.wikipedia.org/wiki/Game_Gear)2P for [MiSTer FPGA](https://mister-devel.github.io/MkDocs_MiSTer/)

This branch is a special version for split screen multiplayer.
In case you are searching for the normal GameGear core, please go here:
https://github.com/MiSTer-devel/SMS_MiSTer

# HW Requirements/Features
SDRAM addon is required.

# Foldername
All Games go to GameGear2P folder. 

You can create a symlink to your SMS or GameGear folder if you want to use the same games.

# Status
Most multiplayer games should be supported and working


# Video Output
A separation line can be enabled in OSD, which will turn the last/first pixel black.

# Audio Output
Selectable in OSD:
- Core 1 to both Channels(left/right)
- Core 2 to both Channels(left/right)
- Mix both cores
- Core 1 to left Channel, Core 2 to right Channel
